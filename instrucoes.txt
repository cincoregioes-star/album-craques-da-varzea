CÁPSULA DO TEMPO DIGITAL — MODELO-BASE OFICIAL

1. OBJETIVO
Este é o molde-mãe para criar novos álbuns digitais interativos com:
- páginas temáticas;
- figurinhas;
- pacotes;
- colagem manual;
- ficha técnica;
- quiz;
- game para ganhar pacotes.

2. ARQUIVO PRINCIPAL DE EDIÇÃO
Edite principalmente:

dados-tema.js

Nele você troca:
- nome do álbum;
- subtítulo;
- textos;
- páginas;
- nomes das figurinhas;
- descrições;
- perguntas do quiz;
- recompensas.

3. FIGURINHAS
Coloque as imagens na pasta:

figurinhas/

Use o padrão:

figurinha-01.webp
figurinha-02.webp
figurinha-03.webp
...
figurinha-72.webp

Também aceita .png, .jpg e .jpeg.

4. ÁUDIOS
Coloque os áudios em:

audio/album/
audio/game/

Sugestão:
audio/album/fundo.mp3
audio/album/abrir-pacote.mp3
audio/album/virar-pagina.mp3
audio/album/colar-figurinha.mp3
audio/album/click.mp3

audio/game/match.mp3
audio/game/vitoria.mp3

Se não colocar áudio, o álbum continua funcionando.

5. GAME
O game fica em:

game/index.html

Quando o usuário vence, volta para o álbum com recompensa:
../index.html?gameReward=1

O álbum reconhece e adiciona +1 pacote.

6. NOVOS TEMAS
Para criar um novo tema:
- copie esta pasta;
- renomeie a pasta;
- altere dados-tema.js;
- troque as figurinhas;
- ajuste cores no style.css se quiser;
- publique no GitHub Pages.

7. TEMAS RECOMENDADOS
- Craques da Várzea Digital

- Debutante

9. OBSERVAÇÃO COMERCIAL
Este modelo pode ser vendido em três versões:
- Simples: fotos + textos + QR Code.
- Interativo: figurinhas + pacotes + colagem.
- Premium: interativo + quiz + game + recompensas.


10. FICHA DETALHADA POR FIGURINHA

A partir desta versão, cada figurinha pode ter uma ficha detalhada própria.

Você pode continuar usando o formato simples:

["Título da Figurinha", "Descrição da figurinha."]

Ou usar o formato profissional:

{
  titulo: "Fundação do Time",
  descricao: "Registro da criação do time.",
  ficha: {
    nome: "Esporte Clube Exemplo",
    categoria: "História do time",
    ano: "1998",
    local: "Beberibe-CE",
    historia: "O time nasceu de encontros de amigos nos fins de semana.",
    curiosidade: "A primeira bola foi comprada por contribuição dos jogadores.",
    mensagem: "Uma história construída por comunidade, amizade e paixão."
  }
}

CAMPOS ÚTEIS POR TEMA:

Futebol amador:
nome, apelido, posicao, camisa, time, temporada, local, historia, curiosidade, conquista, patrocinador, mensagem

Casamento:
titulo, data, local, pessoas, historia, mensagem, audio

Árvore genealógica:
nomeCompleto, parentesco, nascimento, falecimento, local, biografia, historia, mensagem

Escola:
nome, funcao, turma, ano, projeto, local, historia, conquista, mensagem

Empresa:
nome, cargo, setor, ano, projeto, conquista, cliente, historia, mensagem

A ficha detalhada só aparece quando a figurinha for colada.
